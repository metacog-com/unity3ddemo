Detailed Description

Continous pushing of events to the Metacog platform.

This class receive events and store them into batches, that are sent periodically to the Metacog platform.

Store event for later sending to Metacog.

A timestamp and a incremental index are added at queue-insertion time.
The event is serialized into a json string, then pushed into the eventBuffer (memory). if the buffer is full, it will flush into the eventStorage.
