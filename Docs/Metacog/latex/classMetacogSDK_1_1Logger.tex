\subsection{Detailed Description}
Continous pushing of events to the \hyperlink{classMetacogSDK_1_1Metacog}{Metacog} platform. 

This class receive events and store them into batches, that are sent periodically to the \hyperlink{classMetacogSDK_1_1Metacog}{Metacog} platform.
